## Running the App Locally

```
cd [appFolder]
python app.py
```

Frontend files are available at `127.0.0.1:8000`

API is available at `127.0.0.1:8000/api/..`

**Example endpoint**

`127.0.0.1:8000/api/data/app/` should resolve with metadata about the application.
